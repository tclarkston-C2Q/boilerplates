# c2q Centerlight Health Admin Dashboard

Expo + React Native Web admin shell for elderly care operations.

## Quick start

```bash
# Install dependencies
npm install
# or: yarn

# Run Expo dev server
npm start
# or: yarn start
```

Then launch the **web** target from Expo Dev Tools or run:

```bash
npm run web
# or: yarn web
```

This project uses:
- Expo
- react-native-web
- NativeWind (Tailwind)
- Gluestack UI
- react-native-chart-kit
